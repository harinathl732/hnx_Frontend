HNX Quantum - Privacy Policy

1. Data We Collect

We do not collect or monitor your live trading strategy settings or positions. The platform operates on user-controlled REST API mappings.

2. Broker Secret Security

All API secret keys and session tokens are encrypted at the database column level utilizing advanced **AES-256-GCM** algorithms. No credentials are stored in unencrypted plain text.

3. Activity Logging

We keep temporary performance logs strictly to show live P&L data curves inside your local dashboard views.

2. Execution & Network Liability

Algorithmic order execution is subject to third-party broker API uptimes, network latency, and exchange-level outages. We bear no liability for capital losses incurred due to delayed session responses or connectivity timeouts.

3. Automated Stop Limits

Users fully acknowledge that daily MTM Guards are executed based on active background loop checks. Slippage during high-volatility events may cause real execution limits to differ slightly from target thresholds.

Help Hub & Support FAQ

Q: How do I configure my Zerodha credentials?

Go to the **Settings** tab. Paste your developer API Key and API Secret from your Kite Console. Link daily by clicking **Authenticate Session**.

Q: How does the MTM Guard work?

The MTM Guard checks your open P&L every second. If your net profit reaches your daily target, or your net loss breaches your stop limit, it forcefully triggers reversing market exit orders to secure remaining premiums.

Q: Can I run backtests first?

Yes. If your FastAPI server is offline, HNX Quantum launches in local offline simulator mode so you can preview strategy behaviors completely risk-free.
